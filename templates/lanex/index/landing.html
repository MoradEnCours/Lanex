Lanex

What Lanex is about:


• A web application that enables users to connect with one another to exchange languages, helping to improve language knowledge and allow users to create new language buddies along the way
• Users are able to set up language requests based on their languages of interest and preferred locations
• Reqeust can be anything related to language learning: request to meet, have a online chat, join a study group, get recommendation, find a teacher etc.
• Users have the ability to accept language requests created by other users to take part in language-related activities
• Users are granted the facility to make comments on language requests
• Users are provided with a search feature to make it easer to find language requests better geared towards their preferences
